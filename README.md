# dotfiles

Managed with [GNU Stow](https://www.gnu.org/software/stow/).

## Packages

| Package | Contents |
|---------|----------|
| `zsh`   | `.zshrc`, `.zprofile`, `.zshenv` |
| `git`   | `.gitconfig` |
| `nvim`  | `.config/nvim/` (kickstart.nvim) |
| `tmux`  | `.tmux.conf` |

## Bootstrap

```bash
cd ~/.dotfiles
stow zsh git nvim tmux
```
